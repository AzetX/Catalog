Клик на чекбокс, меняет его состояние на true. Товар должен автоматически добавляться
в корзину, если чекбокс имеет состояние - true.
При открытии корзины можем увидеть товары, у которых чекбокс - true. 
При этом состояние чекбокса остается true.
Возможно удилть товар из корзины. При этом состояние чекбокс станет = false.

Варианты:
Когда чекбокс поменял состояние на true при клике, 
то нужно определять id товара, к которому чекбокс принадлежит.
Искать данные о товаре в исходном массиве по id?(или сразу брать из уже имеющегося div?+). Еслии id===13, то
добавле данный товар в корзину, образуя тем самым массив. 
Если хотим удалить, то при клике на кнопку, идет фильтрация ID по полученному раннее массиву и удаляется 
тот id который хотели удалить. При этом стэйт чекбокса становится === false  

Смотрим на product id.
